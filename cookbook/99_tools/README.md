# Tools Cookbook

This directory contains examples and patterns for working with tools in Agno.

## Contents

- `tool_args_delta.py` - Streaming tool call arguments (tool_args.delta) feature

## Tool Args Delta

The `tool_args.delta` feature enables real-time streaming of tool call arguments as they are generated by the model. See:

- **[tool_args_delta.md](tool_args_delta.md)** - Complete documentation
- **[tool_args_delta.py](tool_args_delta.py)** - Working examples

## Quick Start

```python
from agno.agent import Agent
from agno.run.agent import ToolCallArgsDeltaEvent

agent = Agent(
    model=OpenAIChat(id="gpt-4"),
    tools=[my_tool],
    stream_events=True,
)

for chunk in agent.run("Use my_tool", stream=True):
    if isinstance(chunk, ToolCallArgsDeltaEvent):
        print(f"Delta: {chunk.delta}")
```
